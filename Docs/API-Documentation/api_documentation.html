مستندات API - ZB56

این مستندات به شما کمک می‌کند که نحوه استفاده از API‌های موجود در ZB56 را یاد بگیرید.

API 1: ثبت‌نام کاربر

URL: /api/register

متد: POST

پارامترها:

  • username: نام کاربری (رشته)
  • email: ایمیل (رشته)
  • password: رمز عبور (رشته)

پاسخ موفق:

{
            "status": "success",
            "message": "کاربر با موفقیت ثبت شد"
        }

پاسخ خطا:

{
            "status": "error",
            "message": "اطلاعات وارد شده اشتباه است"
        }

API 2: ورود به سیستم

URL: /api/login

متد: POST

پارامترها:

  • username: نام کاربری (رشته)
  • password: رمز عبور (رشته)

پاسخ موفق:

{
            "status": "success",
            "message": "ورود موفقیت‌آمیز"
        }

پاسخ خطا:

{
            "status": "error",
            "message": "نام کاربری یا رمز عبور اشتباه است"
        }
