راهنمای توسعه‌دهندگان - ZB56

این راهنما برای توسعه‌دهندگان است تا نحوه توسعه و افزودن ویژگی‌های جدید به پروژه ZB56 را یاد بگیرند.

راه‌اندازی محیط توسعه

برای شروع توسعه، ابتدا محیط توسعه را با استفاده از Docker راه‌اندازی کنید.

docker-compose up --build

این دستور، محیط Docker را ساخته و آن را راه‌اندازی می‌کند.

افزودن ماژول جدید

برای افزودن ماژول جدید به پروژه، مراحل زیر را دنبال کنید:

  1. یک پوشه جدید در ZB56-Plugins ایجاد کنید.
  2. کدهای مربوط به ماژول را در این پوشه قرار دهید.
  3. ماژول جدید را به فایل __init__.py وارد کنید تا در سیستم شناسایی شود.
